# MoviesApi

This project was generated with [Nest JS].

## Development server

Run `npm run start:dev` for a dev server. Navigate to `http://localhost:3200/`.

# Project Details
This api allows to add user to the system along with adding/removing favourite movies from the users.
MongoDB is used to store the data and mongoose is used to run queries.
This project consists of a restAPI and service which allows to run the requests provided from the front-end.
There is also an attempt done to use postgres with typeORM for retriving and inserting data which available in `user-not-working` folder
